Enterprise Financial & Credit Management System
Full-Stack / Frontend Developer

Project Overview: Developed a comprehensive, scalable Enterprise Resource Planning (ERP) application designed to streamline financial exposure, credit limit tracking, and commission management. Built with a modern React architecture and a Supabase (PostgreSQL) backend, the system centralizes financial operations and provides real-time analytics for enterprise risk management.

Key Responsibilities & Features:

Real-Time Financial Dashboard: Engineered an interactive analytics dashboard visualizing critical KPIs such as Total Active Limits, Total Utilized, Net Outstanding, and Overdue Exposure.
Complex Data Matrix & Analytics: Implemented advanced data aggregation (using Supabase RPCs) to generate Customer-wise Utilization, Bank-wise Summaries, and Customer-Bank Risk Matrices.
Master Data Management: Built robust, secure configuration modules for managing core organizational entities, including Banks, Departments, Companies, Currencies, and Customers.
Transaction Processing: Developed intuitive data entry workflows for accurately tracking Credit Limits and processing Commissions.
Advanced Financial Reporting: Designed comprehensive reporting interfaces (General Ledger, Gross Reports, and Payment Reports) with a dedicated, print-optimized routing architecture for generating clean, paginated financial statements.
Secure Architecture: Implemented session-based authentication and protected routing flows to ensure secure access to sensitive financial records.
Tech Stack:

Frontend: React 19, Vite, React Router DOM, Vanilla CSS
Backend / Database: Supabase (PostgreSQL, RPC Functions, Authentication)
Architecture: Component-driven, RESTful Integration, Print-Optimized UI
